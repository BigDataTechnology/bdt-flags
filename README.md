# yoFlags

AngularJs directive for get flag from country code

## Dependencies
  - Angular ~1.5.0

## Installation
Bower:

```sh
$ bower install yoFlags=http://git.bdt.systems:8888/dev/flags.git --save
```

```javascript
angular.module('appName', [
  'yoFlags'
])
```

## Use

```html
<flag country="us" size="16"></flag>
<flag country="us" size="32"></flag>
```
