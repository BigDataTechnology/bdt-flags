# bdt-flags

Simple Angular directive that return flag icon from country code

## Dependencies
  - Angular ~1.5.0

## Installation
Bower:

```sh
$ bower install bdt-flags --save
```

```javascript
angular.module('appName', [
  'bdt-flags'
])
```

## Use

```html
<flag country="us" size="16"></flag>
<flag country="us" size="32"></flag>
```
